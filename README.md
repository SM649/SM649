## 🧠 About Me

I'm a **Software Engineering graduate** from the National University of Modern Languages (NUML), driven by a deep curiosity about how machines learn, reason, and make sense of the world. I work across **AI/ML, web development, and software quality engineering**, and I'm especially interested in what surrounds the model — the reliability, infrastructure, and architecture that make AI systems actually usable in the real world.

- 🔭 I work on **AI-driven applications**, NLP pipelines, and reliable web systems
- 🌱 Currently going deeper into **RAG, prompt & context engineering, and system architecture**
- 🧪 Background in **SQA & automation testing** (Playwright, CI/CD with GitHub Actions)
- 📝 Published research: *Beyond the Model: The Missing Layer in AI-Driven Systems*
- 🤝 Part of the [@Orenda-Project](https://github.com/Orenda-Project) organization
- 💬 Ask me about **NLP, sentiment analysis, Flask apps, or test automation**
- 📫 Reach me at **saleh72muhammad@gmail.com**
